Alumni Association Platform

The Alumni Association Platform is a full-stack web application designed to connect alumni, students, and administrators on a unified portal. It streamlines alumni management, fosters professional networking, encourages giving back through donations, and promotes community engagement through events and job opportunities.

Features
	•	Authentication System
	•	Secure registration and login for alumni and admins
	•	Role-based access control
	•	Alumni Directory
	•	Browse and search alumni profiles by batch, branch, location, etc.
	•	Job Portal
	•	Alumni and recruiters can post jobs
	•	Students can explore and apply for opportunities
	•	Donation System
	•	Alumni can contribute to their alma mater
	•	Integrated with Razorpay for seamless payment processing
	•	Events Management
	•	Create, manage, and RSVP for events
	•	View upcoming and past alumni events
	•	Notifications
	•	Real-time alerts for events, job openings, and admin updates
	•	Chat System
	•	One-to-one and group messaging for community interaction
	•	Admin Dashboard
	•	Manage users, jobs, events, donations, and announcements

Tech Stack
	•	Backend: Node.js, Express.js
	•	Database: MySQL
	•	Payment Gateway: UPI 
	•	Authentication: JWT-based login system
	•	Frontend: HTML,CSS,JavaScript
